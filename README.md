## 全局设置

- 如果你安装了jQuery Code Snippets插件

    `使用前必须卸载(禁用)jQuery Code Snippets`

- 如果你想禁用开启vscode时自启笔记页(可以通过右键`查询web笔记`)

    `vscodePluginDemo.showTip：false`
